# Plan: Kill Process on Port 8000

## Problem
Port 8000 is occupied by an existing Python process (PID 37280), preventing a new server instance from starting.

## Solution

### Step 1: Kill the existing process
```bash
taskkill /PID 37280 /F
```

### Step 2: Verify port is free
```bash
netstat -ano | findstr :8000
```

### Step 3: Restart the server
```bash
cd backend && python main.py
```

## Files
No code changes needed — this is a runtime issue only.

## Verification
- Server should start without `[Errno 10048]` error
- `http://localhost:8000` should respond
